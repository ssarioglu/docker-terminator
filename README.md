# Terminator Docker Image
Yet another docker image contains;

- Bash
- AWS CLI
- Azure CLI
- Telnet / SSH
- DNS Tools
- Nmap
- Zmap
- MTR
- Hping3
- Redis Cli

Run Docker Container
-----------------

    docker run -it ssarioglu/terminator
